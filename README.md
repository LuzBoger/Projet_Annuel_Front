# GloTrush Frontend

Application React avec TypeScript et Vite.

## Installation

```bash
npm install
```

## Commandes

### Développement
```bash
npm run dev
```

### Tests
```bash
npm run test
npm run test:ui
npm run test:coverage
```

### Build
```bash
npm run build
```

### Linting
```bash
npm run lint
```

### Preview
```bash
npm run preview
```

## Structure

- `/src/pages` : Pages de l'application (Home, Login, Register)
- `/src/components` : Composants réutilisables
- `/src/test` : Configuration des tests
